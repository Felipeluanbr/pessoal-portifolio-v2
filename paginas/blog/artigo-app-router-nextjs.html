Guia definitivo para o App Router no Next.js 13

17 de julho de 2023 — 7 min de leitura

O Roteador de Aplicativos no Next.js é responsável por gerenciar o roteamento do lado do cliente em sua aplicação. Ele fornece uma maneira declarativa e intuitiva de definir rotas, gerenciar a navegação e URLs dinâmicas.

Principais recursos do App Router

  • Roteamento baseado em arquivos: cada página é um arquivo dentro do diretório pages.
  • Rotas dinâmicas: crie rotas com colchetes ([ ]) para parâmetros dinâmicos.
  • Componente Link: navegação do lado do cliente sem recarregar a página.
  • Navegação programática: utilize o hook useRouter para redirecionamentos dinâmicos.
  • Divisão de código: o Next.js carrega apenas o que é necessário para cada página.

Para acompanhar as novidades, confira a documentação oficial do Next.js.
